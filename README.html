Uma calculadora web simples desenvolvida utilizando HTML, CSS e JavaScript. O projeto tem como objetivo praticar conceitos fundamentais de manipulação do DOM, eventos e lógica de programação no JavaScript. Funcionalidades: Adição (+) Subtração (-) Multiplicação (*) Divisão (/) Limpar display (🧹) Apagar último número (⌫) Interface simples e responsiva (📱) */ HTML5 – Estrutura da aplicação CSS3 – Estilização e layout JavaScript – Lógica da calculadora e manipulação do DOM
